# Ronin_Mind_OS

“A Ronin lives by discipline.”

## Introduction
Ronin_Mind_OS is a personal AI-powered Operating System built for discipline, self-mastery, productivity, and mental clarity.

## Core Philosophy
The Ronin mindset is about living without a master, guided only by discipline, inner strength, and self-control. It emphasizes taking full ownership of one's path and decisions.

## Project Description
Ronin_Mind_OS will include:
- Daily journal
- Expense tracker
- Workout planner + calendar
- Books & learning tracker
- Weekly discipline score
- AI Life Coach (OpenAI)
- Supabase database
- Animated dashboard
- Roadmap for future features

## Tech Stack
- Next.js 14
- TypeScript
- TailwindCSS
- Supabase
- OpenAI API
- Framer Motion

## Installation Instructions
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Skdp7/Ronin_Mind_OS.git
   cd Ronin_Mind_OS
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Copy the example environment file:
   ```bash
   cp .env.example .env.local
   ```
   Fill in your Supabase and OpenAI credentials in `.env.local`.

4. **Run locally:**
   ```bash
   npm run dev
   ```

## Project Vision & Roadmap
The vision is to build a digital discipline OS for personal transformation, empowering users to master their habits and mind.
